# ゆずnote 記事ライブラリ

note-projects/yuzu で制作している記事（全50本）を一覧確認するための静的ダッシュボード。姉妹アカウント「はる」のダッシュボードのゆず版。

- カテゴリ（アフターピル／低用量ピル／避妊・恋愛／婦人科・検査／生理・PMS／セルフケア／相談先／読者Q&A）で絞り込み、フリーワード検索が可能（タイトル・本文・ハッシュタグを対象）
- タイトルをクリックすると本文全体とハッシュタグが展開表示される
- ゆずのブランドカラー（コーラルピンク #F4A9A0 系）を基調とした配色
- ライト／ダークモード両対応（右上のボタンで手動切り替え・設定はブラウザに保存）

## 更新方法

1. `note-projects/yuzu/articles/` の記事や `tools/note-poster/articles.json`（ハッシュタグ）を編集する
2. 抽出・ビルドスクリプトで `index.html` を再生成する（記事データは `index.html` 内の `<script id="article-data">` にJSONとして埋め込まれる）
3. 変更を commit & push すると GitHub Pages に自動反映される（公開設定後）

記事本文・画像・投稿ツールの本体は `note-projects/yuzu` フォルダ側で管理している。このフォルダは閲覧用ダッシュボードのみを含む。
